# Angular-nebular
Angular na versão 15 com biblioteca Nebular

- Atalho para gerar componente
  * ng g c nome-do-componente

- instalar o nebular:
  * ng add @nebular/theme.

  * npm i eva-icons @nebular/eva-icons
